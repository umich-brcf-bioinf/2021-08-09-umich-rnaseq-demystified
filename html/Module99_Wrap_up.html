1 Wrapping up RNA-Seq Demystified

1.1 In summary

We hope you now have more familiarity with key concepts, data types, tools, and how they all connect to enable gene expression analysis from bulk RNA-Seq data.

  • Day 1: UNIX, sequence file types, QC, process of producing a count table

  • Day 2: R, DESeq2, QC visualizations, process of differential expression analysis

  • Learning bioinformatic analyses more like a process than a task

  • Training resources/workshops
